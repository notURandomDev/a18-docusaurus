# 客户端安装

## 前提条件

在开始之前，请确保你的开发环境中已安装以下工具：

- **Node.js** （建议版本：v16.x 或更高版本）
- **npm** 或 **Yarn** （包管理工具，随 Node.js 安装 npm，也可选择安装 Yarn）
- **Expo** **CLI** （用于运行和管理 Expo 项目）
- **Git** （用于克隆项目代码）

此外，你需要在手机上安装 **Expo Go** 应用（iOS 或 Android），或者在电脑上配置一个模拟器（如 Android Studio 或 Xcode）。

## 安装步骤

### 1、克隆项目仓库

在终端中运行以下命令，将项目代码克隆到本地：

```bash
git clone https://github.com/notURandomDev/nativewind-a18.git
cd nativewind-a18
```

### 2、安装依赖

进入项目目录后，使用以下命令安装所有必要的依赖：

<Tabs groupId="package-managers">
  <TabItem value="npm" label="npm">

    ```bash
     npm install
    ```

  </TabItem>
  <TabItem value="yarn" label="yarn">

    ```bash
    yarn install
    ```

  </TabItem>
</Tabs>

### 3、配置 Expo CLI

如果你尚未全局安装 Expo CLI，可以通过以下命令安装：

<Tabs groupId="package-managers">
  <TabItem value="npm" label="npm">

    ```bash
     npm install -g expo-cli
    ```

  </TabItem>
  <TabItem value="yarn" label="yarn">

    ```bash
    yarn global add expo-cli
    ```

  </TabItem>
</Tabs>

:::tip

如果你遇到权限问题，可能需要使用 sudo（Mac/Linux）或以管理员身份运行终端（Windows）。

:::

### 4、启动项目

在项目目录中运行以下命令启动开发服务器：

```bash
npx expo start
```

或者使用快捷命令：

```bash
npm start
```

启动后，你会看到一个二维码。可以用手机上的 **Expo Go** 应用扫描二维码，或者按以下提示在模拟器上运行：

- 按 a 在 Android 模拟器上运行（需安装 Android Studio）。
- 按 i 在 iOS 模拟器上运行（需安装 Xcode，仅限 macOS）。

## 故障排查

### 依赖安装失败

尝试删除 node_modules 文件夹和 package-lock.json 文件，然后重新运行 npm install。

### 二维码无法扫描

确保手机和电脑在同一网络中，或尝试通过 localhost 手动连接。

### 模拟器无法运行

检查模拟器是否正确配置，并确保相关开发工具（Android Studio 或 Xcode）已安装。
